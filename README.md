 Project Description

The Expense Tracker is a user-friendly web interface built with React.js that allows users to interact with the Expense Tracker system.

The frontend focuses on:
- Easy expense entry
- Clear expense visualization
- Smooth communication with the backend APIs

Using React Context API, the application manages global authentication and expense state efficiently without using external state management libraries.

----------------------------------------------------------------------------------------------

 Technologies Used

- React.js – Component-based UI framework
- React Router DOM – Client-side routing
- Axios – API communication
- Context API – Global state management
- Netlify – Frontend deployment

----------------------------------------------------------------------------------------------

Real-World Use Cases

- Personal budgeting and expense tracking
- Group expense management (trips, teams, families)
- Financial awareness and planning
- Small-scale finance applications
